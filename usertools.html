User Tools

Galleon

Galleon is a deployment of Tezori, an open source wallet framework for Tezos, supporting both software and hardware wallets in eight language on Mac, Windows, and Linux.

Learn more about Galleon

Arronax

Conventional block explorers are quite constrained and only useful for people looking for specific things.

We are building Arronax, a blockchain analytics tool which allows full-on exploration of blockchain data for open-ended searches by analysts, developers, traders, stakers and other users.

Read about the vision for Arronax

Mininax

Mininax was a fun two-week summer 2019 project which served as a retro minimalist tribute to the early web and the day on Minitel. The purpose of the project was to explore full-on functional programming using ReasonML and also to show the power of the Crytonomic tool stack.
