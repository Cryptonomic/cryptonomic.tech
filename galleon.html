Galleon

A Smart Wallet for Tezos

Galleon is a software wallet for the Tezos blockchain. It is a deployment of Tezori (T2), an open source desktop wallet framework for Tezos, which runs on Cryptonomic infrastructure. Galleon supports hardware signing and is translated into eight languages with releases available for Mac, Windows and Linux.
