Conseil

Conseil is a multi-chain industrial-grade blockchain indexer written in Scala, currently focused on Tezos. Conseil collects blockchain data and stores it for easy querying and retrieval. Along with ConseilJS, it forms a spine for building permissioned and decentralized blockchain apps.

A technical overview and documentation for Conseil can be found on the GitHub repo.
